We are using Node.js , Chromium , and Electron .

This is the page window. You can move this element around in the index.html for when whoever starts working on the ui. To do so, I made it easy for you. Just move the div with the page-window class on it to anywhere in your new html structure within index.html and it should work. It imports the html for the page into index.html and it imports its own css and js. Basically just an encapsulated page. I would advise you guys to build your stuff in a similar fassion just to have a healthy separation of functionality.

To achieve this html import method, create an object similar to mine and swap out the file with your own. It needs to have the 'import' class but it doesnt need to be wrapped in it's own div. I just did that to size it properly. Then, in your own html that you want to import into index.html, include your necessary css and js. Feel free to reference mine.
